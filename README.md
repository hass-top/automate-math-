## Installation

with your terminal 
```git 
git clone https://github.com/hass-top/automate-math-.git

cd automate-math-/main_project/

python3 -m venv env
```
On Windows:
```git 
  env\Scripts\activate
```
On macOS/Linux:
```git 
  source env/bin/activate

pip install -r requirements.txt

cd automate-math-/main_project/

```
and then execute the app 
```git 
python3 main.py
```
